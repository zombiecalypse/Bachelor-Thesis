\frame{
	\frametitle{Semantic Function}
	\pause
	The {\em semantic function}\/ translates between the language $L$, we use to 
	describe a program and the mathematical objects $M$, it describes.

	\pause
	For example:
	\begin{equation*}
		\interpret[\lambda \text{-calculus}]{\lambda x.\,x}(f) = id(f) = f
	\end{equation*}
	\pause
	If given expricitly it gives a specification for the programming language.
}
